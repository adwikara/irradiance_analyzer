NSRDB Daily Irradiance

Aditya Wikara

How To Use:

1. Data is taken from NSRDB (National Solar Radiation Database)

2. Only US locations are available, any international coordinates will not work

3. After providing the correct inputs, press the get data button.

4. It takes 5-10 seconds for the backend to generate the solar radiation table

5. The table will present the hourly solar radiation data for the current day.

6. Click the refresh button to reset the table and the inputs.
